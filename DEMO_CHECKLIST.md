## Post-Demo Q&A

### Expected Questions

**Q: How long does it take?**
> A: 30-60 seconds per lead for real-time, or batch 100 leads overnight.

**Q: What if the data is wrong?**
> A: The actor extracts from public sources. You can configure which sources to use and set confidence thresholds. AI scoring is 80%+ accurate.

**Q: Can I customize the fields?**
> A: Yes, fully customizable input/output field mapping. Pick exactly what you need.

**Q: What about privacy/GDPR?**
> A: Only public data is collected. No personal data stored by the actor. All data goes directly to your Airtable.

**Q: How much does it cost?**
> A: ~$0.003 per lead. $3 per 1,000 leads. $30 per 10,000 leads. 99%+ cheaper than alternatives.

**Q: Can I use it with other CRMs?**
> A: Currently Airtable-native, but we have an API mode for integrating with any system via webhooks.

**Q: What about bulk enrichment?**
> A: Set up scheduled runs to enrich 100 leads at a time, or use the button for ad-hoc enrichment.
