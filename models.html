AI Model Selection

Choose the right AI model for lead scoring based on your needs.

Quick Recommendations

Use Case Model Provider Why
Best Overall Claude 3 Haiku Anthropic Fast, affordable, quality results
High Volume GPT-4o Mini OpenAI Cost-effective for large batches

OpenAI Models

GPT-4o (Recommended for Production)

gpt-4o

  • Latest production model with structured output support
  • Best for enterprise deployments
  • Auto JSON mode with graceful fallback

GPT-4o Mini (Budget Option)

gpt-4o-mini

  • 10x cheaper than GPT-4o
  • Good for high-volume processing (1000+ leads/day)
  • Faster response times

GPT-3.5 Turbo (Balanced)

gpt-3.5-turbo

  • Good balance of cost and performance
  • Reliable for most use cases

Anthropic Models

Claude 3 Haiku (Top Pick)

claude-3-haiku-20240307

  • Excellent performance across all metrics
  • Fast responses (~3 seconds)
  • Affordable pricing
  • 200K context window

Claude Haiku 4.5 (Latest)

claude-haiku-4-5-20251001

  • Newest model from Anthropic
  • More conservative scoring than Claude 3
  • Enhanced safety features
Note: Claude 3.5 Sonnet and Claude 4.5 Opus are also available via API. Claude 3 Haiku is recommended for this use case based on performance testing.

AWS Bedrock Models

Bedrock: Claude Haiku 4.5 (AWS Native)

us.anthropic.claude-haiku-4-5-20251001-v1:0

  • Cheapest option available (~50% less than direct API)
  • Native AWS integration (IAM, CloudWatch, VPC)
  • Global cross-region inference
  • Best for AWS-first organizations
Coming Soon: Support for Meta Llama, Amazon Nova, and Mistral models on Bedrock requires different API integration. Check back for updates.

Configuration Examples

OpenAI (GPT-4o)

{
  "llmScoring": {
    "enabled": true,
    "provider": "openai",
    "apiKey": "sk-...",
    "model": "gpt-4o",
    "icpCriteria": "B2B SaaS companies, 10-500 employees"
  }
}

Anthropic (Claude 3 Haiku)

{
  "llmScoring": {
    "enabled": true,
    "provider": "anthropic",
    "apiKey": "sk-ant-...",
    "model": "claude-3-haiku-20240307",
    "icpCriteria": "B2B SaaS companies, 10-500 employees"
  }
}

AWS Bedrock (Claude)

{
  "llmScoring": {
    "enabled": true,
    "provider": "bedrock",
    "model": "us.anthropic.claude-haiku-4-5-20251001-v1:0",
    "region": "us-east-1",
    "apiKey": "your-bedrock-api-key",
    "icpCriteria": "B2B SaaS companies, 10-500 employees"
  }
}

Get API Keys:

  • OpenAI: platform.openai.com/api-keys
  • Anthropic: console.anthropic.com/settings/keys
  • AWS Bedrock: AWS Bedrock Setup Guide
