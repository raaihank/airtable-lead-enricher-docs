# Extension Screenshots

This directory contains screenshots for the extension documentation.

## How to Capture Screenshots

### 1. Welcome Screen
- Open extension in Airtable
- Capture the welcome screen
- Save as: `01-welcome-screen.png`

### 2. Step 1 - API Keys
- Click "Get Started"
- Fill in API keys (use fake/masked values)
- Capture the screen
- Save as: `02-step1-api-keys.png`

### 3. Step 2 - Field Mapping
- Click "Continue" from Step 1
- Show the field mapping interface
- Click "Auto-fill" to show matched fields
- Save as: `03-step2-field-mapping.png`

### 4. Step 3 - Select & Enrich
- Click "Continue" from Step 2
- Show table/view selection
- Show filtered records count
- Save as: `04-step3-select-enrich.png`

### 5. Enrichment Progress
- After clicking "Enrich X Records"
- Show progress bar and status
- Save as: `05-enrichment-progress.png`

### 6. Completed Results
- After enrichment completes
- Show completed status with link to console
- Save as: `06-enrichment-complete.png`

## Screenshot Specifications

- **Format**: PNG
- **Width**: 1200-1400px recommended
- **DPI**: 144 (retina) or 72 (standard)
- **Blur sensitive data**: Use blur tool for API keys, record data
- **Annotations**: Optional arrows/highlights for key features
